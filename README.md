# Calypso 🎵
**Experience music in its purest form.**

Calypso is a modern, high-quality music downloader for Android. It combines a powerful search engine with a state-of-the-art **Material 3 Expressive** interface, designed to be fast, clean, and beautifully integrated with the latest Android standards.

---

## ✨ Features

- 🔍 **Smart Search:** Find any track instantly with a fluid and responsive search engine.
- ⚡ **Direct Downloads:** High-speed MP3 downloads (320 kbps) powered by optimized servers.
- 🎨 **Material 3 Expressive:** A stunning UI featuring ultra-rounded corners (36dp), tonal surfaces, and full support for **Dynamic Colors (Material You)**.
- 🌊 **Fluid Motions:** Every interaction feels alive thanks to Spring Animations and the Material motion system.
- 📂 **Custom Storage:** Full support for Scoped Storage (SAF), allowing you to choose exactly where your music goes.
- 🌍 **Multilingual:** Fully translated into English, Spanish, and Portuguese.
- 🚀 **Update System:** Stay current with a built-in version checker.

## 🛠 Tech Stack

- **Language:** Java
- **UI:** Material Components 3 (M3)
- **Networking:** Retrofit 2 + OkHttp 3
- **Image Loading:** Glide 4
- **Animations:** Spring Animation (DynamicAnimation) & Android View Animations
- **Architecture:** ViewBinding & Modern Android Best Practices

## 📥 Installation

1. Go to the [Releases](https://github.com/20Josy07/Calypso/releases) section.
2. Download the latest `Calypso-v1.0.apk`.
3. Enable "Unknown Sources" in your Android settings if required.
4. Install and enjoy!
