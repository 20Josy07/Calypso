# Calypso 🎵
**Experience music in its purest form.**

Calypso is a modern music downloader for Android that combines powerful search and download capabilities with a state-of-the-art **Material 3 Expressive** interface. Designed to be fast, clean, and beautifully integrated with the latest Android standards.

## ✨ Key Features
- 🔍 **Smart Search:** Find any track instantly with a fluid and responsive search engine.
- ⚡ **Direct Downloads:** High-speed MP3 downloads (320 kbps) using optimized free servers.
- 🎨 **Material 3 Expressive:** A stunning UI with ultra-rounded corners, tonal surfaces, and dynamic colors (Material You).
- 🌊 **Fluid Motions:** Powered by Spring Animations and Material motion system for a "breathing" UI experience.
- 📂 **Custom Storage:** Full support for Scoped Storage (SAF), allowing you to choose your preferred download folder.
- 🌍 **Multilingual:** Available in English, Spanish, and Portuguese.
- 🚀 **Update System:** Built-in version checker to keep you always up to date.

## 🛠 Tech Stack
- **Language:** Java / Android SDK
- **Architecture:** MVVM / ViewBinding
- **Networking:** Retrofit 2 + OkHttp 3
- **Image Loading:** Glide 4
- **Animations:** Spring Animation (DynamicAnimation)
- **UI:** Material Components 3 (M3)

## 📥 Installation
1. Download the latest APK from the [Releases](https://github.com/josyam/Calypso/releases) section.
2. Enable "Unknown Sources" in your device settings.
3. Install and start your journey!
